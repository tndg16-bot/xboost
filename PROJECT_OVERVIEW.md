### コンセプト
- **対象**: クリエイター、マーケター、起業家
- **課題**: X運用の継続性、効率化、改善
- **解決策**: AI支援、自動化、分析、予約投稿の統合プラットフォーム

## 🏗️ システムアーキテクチャ

### 技術スタック
```
Frontend: Next.js 16.1.4 + React 19 + TypeScript 5
Styling: Tailwind CSS 4 + shadcn/ui
Backend: Next.js API Routes (Edge Runtime対応)
Database: PostgreSQL + Prisma 7.3.0
Auth: NextAuth 4.24.13 (Twitter OAuth 2.0)
Payment: Stripe (サブスクリプション)
AI: OpenAI API (GPT-4)
External: Twitter API v2
```

### ディレクトリ構成
```
xboost/
├── apps/web/                  # Next.jsアプリケーション
│   ├── app/                   # App Router
│   │   ├── api/              # API Routes (20+ endpoints)
│   │   ├── (pages)/          # ページコンポーネント
│   │   └── layout.tsx        # ルートレイアウト
│   ├── components/           # Reactコンポーネント
│   ├── lib/                  # ユーティリティ
│   │   ├── auth.ts          # 認証設定
│   │   ├── prisma.ts        # DBクライアント
│   │   ├── stripe-server.ts # Stripe統合
│   │   └── twitter-client.ts # Twitter API
│   └── prisma/
│       ├── schema.prisma    # DBスキーマ
│       └── migrations/      # マイグレーション
├── packages/                 # モノレポパッケージ
│   ├── features/            # 機能モジュール
│   │   ├── ai-rewrite/     # AI文章改善

│   │   ├── automation/     # 自動化機能
│   │   └── scheduling/     # 予約投稿
│   └── ui/                  # UIコンポーネント
└── docs/                    # ドキュメント
```
