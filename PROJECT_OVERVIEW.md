## 🎯 プロジェクトビジョン

> **「発信をプロダクトが売れる導線に」**
> 発信は才能じゃない、フォーマットだ

### コンセプト
- **対象**: クリエイター、マーケター、起業家
- **課題**: X運用の継続性、効率化、改善
- **解決策**: AI支援、自動化、分析、予約投稿の統合プラットフォーム

### 実績データ（開発者本人）
- 3週間で1,000万インプレッション達成者複数
- 1日500〜600フォロワー増加（月約1万ペース）

## 🏗️ システムアーキテクチャ

### 技術スタック
```
Frontend: Next.js 16.1.4 + React 19 + TypeScript 5
Styling: Tailwind CSS 4 + shadcn/ui
Backend: Next.js API Routes (Edge Runtime対応)
Database: PostgreSQL + Prisma 7.3.0
Auth: NextAuth 4.24.13 / Auth.js (Twitter OAuth 2.0)
Payment: Stripe (サブスクリプション)
AI: OpenAI API (GPT-4)
External: Twitter API v2
```

### ディレクトリ構成
```
xboost/
├── apps/web/                  # Next.jsアプリケーション
│   ├── app/                   # App Router
```
