# 進捗報告テンプレート

---

## 基本テンプレート

```markdown
## 進捗報告
エージェント: [A/B/C]
タスクID: [タスクID]
アクション: [開始/完了/一時停止/ブロッカー発生]
詳細: [任意の詳細情報]
```

---

## 使用例

### タスク開始時

```markdown
## 進捗報告
エージェント: C
タスクID: C-1
アクション: 開始
詳細: TypeScript構文エラーの修正を開始しました。まずcta-checker/page.tsxを確認します。
```

### タスク完了時

```markdown
## 進捗報告
エージェント: C
タスクID: C-1
アクション: 完了
詳細: cta-checker/page.tsxのSVGパスを修正しました。
- Line 189のSVGパスを正しいアイコンに置換
- npm run typecheck でエラーなしを確認
- ファイル: xboost/apps/web/app/cta-checker/page.tsx
```

### ブロッカー発生時

```markdown
## 進捗報告
エージェント: C
タスクID: C-2
アクション: ブロッカー発生
詳細: openaiパッケージのインストールでエラーが発生しました。

エラーメッセージ:
```
npm ERR! code ENOENT
npm ERR! syscall open
npm ERR! path C:\Users\chatg\package.json
```

試したこと:
1. `cd xboost/apps/web` を実行
2. `npm install openai` を実行

PMからのアドバイスを待っています。
```

### 進捗更新（中間報告）

```markdown
## 進捗報告
エージェント: C
タスクID: C-2
アクション: 進捗更新
詳細: ai-pattern-analyzer.tsの修正中です。

完了したこと:
- Line 300: 括弧の修正
- Line 302-306: 変数宣言の追加

残り:
- Line 311, 359の構文エラー修正
- OpenAI APIの型エラー修正

予想完了時刻: 1時間後
```

### 一時停止時

```markdown
## 進捗報告
エージェント: A
タスクID: A-1
アクション: 一時停止
詳細: APIの実装を一時停止しています。

理由: 環境変数の確認が必要です。
- NEXTAUTH_SECRET
- OPENAI_API_KEY

次回セッションで再開予定。
```

---

## アクションタイプ

| アクション | 使用タイミング |
|----------|---------------|
| 開始 | 新しいタスクを開始した時 |
| 完了 | タスクが完了した時 |
| 進捗更新 | タスクの途中経過を報告する時 |
| 一時停止 | タスクを一時的に中断する時 |
| 再開 | 中断したタスクを再開する時 |
| ブロッカー発生 | 解決できない問題が発生した時 |

---

## タスクID一覧

### エージェントA

| ID | タスク名 |
|----|---------|
| A-1 | AIプロフィール関連APIの実装 |

### エージェントB

| ID | タスク名 |
|----|---------|
| B-1 | Stripe決済システムのテスト |
| B-2 | UIコンポーネントのリンク確認 |

### エージェントC

| ID | タスク名 |
|----|---------|
| C-1 | TypeScript構文エラー修正（cta-checker） |
| C-2 | TypeScript構文エラー修正（ai-pattern-analyzer） |
| C-3 | openaiパッケージの追加 |
| C-4 | Lintエラーの解消 |
| C-5 | 共通ユーティリティの実装 |
| C-6 | ドキュメントの更新 |

---

## 詳細報告に含めるべき情報

### タスク完了時

- 修正したファイルのパス
- 行番号（可能な場合）
- 変更内容の要約
- 検証結果（コマンド実行結果など）
- 関連するドキュメント（作成/更新した場合）

### ブロッカー発生時

- エラーメッセージ（全文）
- 試したこと
- 再現手順
- 環境情報（OS、Nodeバージョンなど）

### 進捗更新時

- 完了した項目（リスト）
- 残りの項目（リスト）
- 予想完了時刻（可能な場合）
