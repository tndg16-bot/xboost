# Xboost

> 発信をプロダクトが売れる導線に

X（旧Twitter）運用に必要な全作業を一元管理できるSaaSツール

## 🚀 Features

- **投稿作成**: AI支援、下書き、テンプレート
- **予約投稿**: 2ヶ月先まで予約可能
- **分析・検証**: 1年分のデータ同期、勝ちパターン発見
- **自動化**: バズ投稿の自動再投稿
- **複数アカウント管理**: ワンクリック切り替え

## 📈 実績

- 3週間で1,000万インプレッション達成者複数
- 1日500〜600フォロワー増加（月約1万ペース）

## 👁️ 進捗の見方

### Antigravity様（ユーザー）は、以下の手順で進捗を確認してください：

1. **USER_DASHBOARD.md を開く** - プロジェクト全体の状況が一目でわかります
2. **「全体進捗」を確認** - プログレスバーで現在の進捗を確認
3. **「エージェント状況」を確認** - 各エージェントが何をしているか確認
4. **「ブロッカー」を確認** - 何がプロジェクトをブロックしているか確認
5. **「今日の重点タスク」を確認** - 今日最優先でやるべきことを確認

### 重要な通知があった場合：

- **ブロッカー発生**: 即座にPM（Sisyphus）に連絡してください
- **タスク完了**: 次のタスクを確認してください
- **マイルストーン達成**: プロジェクト全体の状況を確認してください

## 🔗 現状の連携方法

現在、プロジェクトは以下の仕組みで連携しています：

```
Antigravity（ユーザー） → チャット → Sisyphus（PM） → チャット → エージェントA/B/C → ドキュメント更新
```

**特徴**:
- ✅ PMが一元管理しているため、進捗が見やすい
- ✅ ドキュメントが詳細に記録されている
- ⚠️ 手動連携のため、リアルタイム性が低い
- ⚠️ エージェントからの報告がチャットベース

**詳細**: [AGENT_COLLABORATION_SYSTEM.md](./AGENT_COLLABORATION_SYSTEM.md) を参照してください。
