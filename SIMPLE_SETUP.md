## 🚀 3ステップで完了！（5分）

### ステップ1: 2つのファイルをコピー

```bash
# プロジェクトのディレクトリに移動
cd /path/to/your/project

# このガイドとテンプレートをコピー
cp /path/to/xboost/SIMPLE_SETUP.md .
cp /path/to/xboost/templates/PROGRESS_REPORT_TEMPLATE.md .
```

**これだけ！** 次のステップに進んでください。

---

### ステップ2: エージェントを起動

```javascript
// エージェントAを起動
task({
  description: "タスク",
  prompt: "あなたはエージェントAです。\n\nタスク: [タスクの説明]\n\n進捗報告: 1時間ごとに報告してください。",
  subagent_type: "general",
  tools: { "background_task": false }
})
```

### ステップ3: 進捗を確認

エージェントから報告が来たら、このファイルの「進捗記録」に書いてください。

## 📝 進捗報告の例

エージェントから来た報告を、そのままここに貼り付けてください：

```markdown
## 進捗報告
エージェント: A
タスクID: A-1
アクション: 完了
詳細: タスク完了しました。
```

## 🎯 複数のエージェントを起動する場合

```javascript
// エージェントA
task({
  description: "タスクA",
  prompt: "あなたはエージェントAです。\n\nタスク: [タスク]",
  subagent_type: "general",
  tools: { "background_task": false }
})

// エージェントB（並列実行）
task({
  description: "タスクB",
  prompt: "あなたはエージェントBです。\n\nタスク: [タスク]",
  subagent_type: "general",
  tools: { "background_task": false }
})

// エージェントC（並列実行）
task({
  description: "タスクC",
  prompt: "あなたはエージェントCです。\n\nタスク: [タスク]",
  subagent_type: "general",
  tools: { "background_task": false }
})
```
